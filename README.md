# WB Воронка продаж — Аналитический дашборд

Инструмент для анализа воронки продаж WildBerries. Загружайте еженедельные отчёты и отслеживайте динамику CTR, конверсий и выручки.

## Возможности

- 📂 Загрузка файлов «Воронка продаж» из кабинета WB (Excel)
- 📊 Три вкладки: Сводка, Графики, По товарам
- 📅 Поддержка нескольких недель — динамика строится автоматически
- 🔢 Выбор метрик через чипы (CTR, конверсии, заказы и др.)
- 🔍 Фильтр по артикулам с мультиселектом
- ↔️ Перетаскивание столбцов в таблице
- 📈 Дельта: для %-метрик показывает абсолютную разницу (пп), для числовых — относительную (%)
- 🌙 Поддержка тёмной темы

## Структура файлов

```
wb-funnel/
├── index.html   — разметка приложения
├── styles.css   — все стили
├── app.js       — логика (парсинг, фильтры, графики)
└── README.md
```

## Запуск локально

Просто откройте `index.html` в браузере — никакого сервера не нужно.

Или через `live-server` для удобства:
```bash
npx live-server
```

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub (например `wb-funnel`)
2. Загрузите все файлы:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/wb-funnel.git
git push -u origin main
```
3. В настройках репозитория → **Settings → Pages**
4. Source: `Deploy from a branch` → ветка `main` → папка `/ (root)`
5. Сохраните — через минуту сайт будет доступен по адресу:
   `https://ВАШ_ЛОГИН.github.io/wb-funnel`

## Формат файла WB

Приложение читает файл «Воронка продаж» из WB Аналитики. В файле должны быть листы:
- **Общая информация** — период отчёта
- **Фильтры** — сводные данные по всем товарам
- **Товары** — детальная таблица по каждому артикулу

## Поддерживаемые метрики

| Метрика | Тип дельты |
|---|---|
| CTR, % | Абсолютная (пп) |
| Конверсия в корзину, % | Абсолютная (пп) |
| Конверсия в заказ, % | Абсолютная (пп) |
| % выкупа | Абсолютная (пп) |
| Показы | Относительная (%) |
| Переходы в карточку | Относительная (%) |
| Заказали, шт | Относительная (%) |
| Выкупили, шт | Относительная (%) |
| Сумма заказов, ₽ | Относительная (%) |
| Средняя цена, ₽ | Относительная (%) |
| и др. | |
